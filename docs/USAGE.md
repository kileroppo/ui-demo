## 常见使用场景

### 我是设计师

**目标：** 寻找设计灵感，确定项目视觉方向。

**推荐流程：**

1. 打开首页，浏览精选风格卡片，感受各种风格的视觉差异
2. 点击感兴趣的风格卡片，进入详情页查看完整的 CSS 实时演示
3. 在详情页中关注以下信息：
   - **颜色方案** - 主色、辅色搭配建议
   - **适用场景** - 这种风格最适合什么类型的产品
   - **技术参数** - 性能等级、无障碍等级
4. 找到心仪风格后，复制 AI 提示词生成更多设计变体

**快速技巧：**
- 使用 `Cmd/Ctrl + K` 快速打开搜索
- 输入"暗色"、"圆角"、"渐变"等关键词筛选风格
- 在风格库页面使用筛选面板按类别浏览

### 我是前端开发者

**目标：** 确定技术方案，了解各风格的实现复杂度。

**推荐流程：**

1. 访问风格库页面（`/styles`），使用筛选面板按性能等级排序
2. 关注风格详情中的技术参数：
   - **性能等级** - High/Medium/Low，影响页面加载速度
   - **无障碍等级** - AA/AAA，影响 WCAG 合规
   - **CSS 复杂度** - 通过实时演示代码评估实现难度
3. 查看推荐产品了解该风格在哪些类型的产品中被广泛使用
4. 复制 AI 提示词，让 AI 辅助生成具体的 CSS/组件代码

**快速技巧：**
- 在筛选面板中选择 "High Performance" 筛选高性能风格
- Dashboard 类别的风格通常更适合数据密集型应用
- 查看实时演示的 CSS 属性了解核心实现要点

### 我是产品经理

**目标：** 为产品选择合适的设计风格，与设计师沟通设计方向。

**推荐流程：**

1. 直接访问产品推荐页面（`/products`）
2. 找到你的行业分类（科技 SaaS、电商零售、金融保险等）
3. 在对应行业中找到你的产品类型
4. 查看该产品类型推荐的主要风格和备选风格
5. 点击风格名称跳转到详情页，了解视觉效果
6. 将 AI 提示词分享给设计师作为设计方向参考

**快速技巧：**
- 产品页面按行业分组，可以直接定位到你的行业
- 每个产品卡片上显示推荐风格的主色调预览
- 可以对比多个推荐风格，选择最符合品牌调性的

## 搜索技巧

### 中文关键词

系统支持常用中文别名搜索：

| 输入 | 匹配风格 |
|------|----------|
| 玻璃 | Glassmorphism（玻璃拟态） |
| 新拟态 | Neumorphism（新拟物） |
| 暗色/暗黑 | Dark Mode（暗黑模式） |
| 极简 | Minimalism（极简主义） |
| 赛博朋克 | Cyberpunk（赛博朋克） |
| 扁平 | Flat Design（扁平设计） |
| 野蛮 | Brutalism（野蛮主义） |
| 复古 | Retro Futurism（复古未来） |
| 粘土 | Claymorphism（粘土拟态） |
| 极光 | Aurora（极光风格） |

### 英文关键词

直接输入风格的英文名称或部分关键词：

- `glass` - 匹配 Glassmorphism
- `dark` - 匹配 Dark Mode
- `brutal` - 匹配 Brutalism、Neubrutalism
- `retro` - 匹配 Retro Futurism
- `bento` - 匹配 Bento Grid

### 组合搜索建议

搜索框支持以下策略提升搜索效率：

1. **直接搜风格名** - 中文或英文都支持
2. **搜关键特征** - 如"渐变"、"阴影"、"3D"
3. **利用搜索建议** - 输入后等待下拉建议列表
4. **利用历史搜索** - 点击搜索框查看最近搜索记录
5. **键盘快速操作** - 用 ↑↓ 选择建议，Enter 确认

### 搜索快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl + K` | 从任何页面快速聚焦搜索框 |
| `↑` / `↓` | 在搜索建议列表中上下移动 |
| `Enter` | 选择当前高亮的建议项 |
| `Esc` | 关闭建议列表 / 取消搜索 |

## 筛选策略

### 按性能选择

风格库页面的筛选面板支持按性能等级筛选：

- **High Performance** - CSS 实现简单，渲染性能好，适合移动端优先的项目
- **Medium Performance** - 使用了一些高级 CSS 特性（如 backdrop-filter），在现代浏览器上表现良好
- **Low Performance** - 包含复杂动画或大量图层叠加，建议只用于桌面端或展示型页面

**建议：**
- 电商类产品优先选择 High Performance 风格
- 品牌展示页面可以选择 Medium/Low Performance 但视觉冲击力更强的风格
- 移动端应用建议限定在 High/Medium Performance 范围

### 按无障碍等级选择

- **AAA** - 最高无障碍标准，对比度和可读性最佳，适合政府、教育、医疗类产品
- **AA** - 满足基本无障碍要求，适合大部分商业产品
- **A** - 基本对比度，可能需要额外调整以满足合规要求

**建议：**
- 面向公众的政务/教育产品，建议选择 AAA 等级
- 商业 SaaS 产品，建议至少选择 AA 等级
- 纯展示型或艺术类项目可以适当放宽

### 按行业选择

访问产品推荐页面，系统已按 8 大行业预分组：

| 行业 | 典型产品 | 推荐风格方向 |
|------|----------|--------------|
| 科技 SaaS | CRM、项目管理、开发工具 | 极简、扁平、暗色 |
| 电商零售 | 商城、品牌官网 | 鲜明色彩、卡片式 |
| 医疗健康 | 健康 App、医疗平台 | 干净、高对比度 |
| 金融保险 | 银行、投资、保险 | 稳重、专业、高可信度 |
| 教育文化 | 在线课程、知识库 | 友好、易读、色彩温和 |
| 生活服务 | 餐饮、旅行、本地生活 | 活泼、图片丰富 |
| 创意设计 | 设计工具、Portfolio | 前卫、视觉冲击力 |
| 其他 | 更多类型 | 因产品而异 |

## AI 提示词使用方法

### 支持的 AI 工具

本项目生成的提示词适用于以下 AI 工具：

- **ChatGPT (GPT-4/4o)** - 生成设计描述、代码
- **Claude** - 生成设计方案、组件代码
- **通义千问** - 中文场景下的设计建议
- **Midjourney / DALL-E** - 生成设计参考图（需适当调整提示词格式）
- **Cursor / GitHub Copilot** - 辅助生成组件代码

### 如何复制提示词

1. 在风格详情页找到 "中文 AI 提示词" 区域
2. 点击复制按钮（带有复制图标）
3. 按钮会显示"已复制"确认动效
4. 打开你的 AI 工具，直接粘贴使用

### 如何修改提示词获得更好效果

复制到的提示词是通用模板，你可以根据需要修改：

**添加具体约束：**
```
[粘贴的提示词]

额外要求：
- 主色调使用品牌色 #1a73e8
- 适配移动端 375px 宽度
- 使用 Tailwind CSS 实现
- 包含暗色模式变体
```

**指定具体组件：**
```
[粘贴的提示词]

请基于这个风格，为我设计一个：
- 登录页面
- 数据仪表盘
- 产品定价卡片
- 导航栏组件
```

**组合多种风格：**
```
主要风格：[风格A的提示词]
辅助风格：[风格B的提示词]

请融合这两种风格的特点，设计一个现代感的 SaaS 产品首页。
```

**生成参考图片（Midjourney 用）：**
```
UI design, web application, [风格关键词],
modern interface, clean layout,
high quality, 4k, detailed --ar 16:9
```

## FAQ 常见问题

### Q: 风格数据多久更新一次？

当前版本包含 76 种风格和 161 种产品类型。数据来源于 UI/UX Pro Max 技能库，会随着上游数据更新而更新。

### Q: 可以离线使用吗？

可以。运行 `pnpm build` 构建后，`dist/` 目录下是纯静态文件，可以直接用任何 HTTP 服务器托管，不需要后端服务。

### Q: 搜索没有结果怎么办？

- 尝试用英文关键词搜索（如 "glassmorphism" 代替 "玻璃拟态"）
- 减少搜索关键词长度，使用更通用的词
- 使用筛选面板代替搜索来浏览所有风格
- 查看搜索建议中的热门推荐

### Q: 如何判断一种风格是否适合我的项目？

参考以下维度：
1. **行业匹配** - 在产品推荐页面查看你的行业推荐
2. **性能要求** - 移动端优先选 High Performance
3. **用户群体** - 面向老年人/残障人士选择 AAA 无障碍等级
4. **品牌调性** - 科技感选赛博朋克/暗色，亲和力选圆角/渐变

### Q: AI 提示词生成的效果不理想怎么办？

- 在提示词后追加具体约束（颜色、尺寸、用途）
- 指定具体的组件类型而不是整页设计
- 尝试组合两种风格的提示词
- 在提示词开头加上"你是一个资深 UI 设计师"的角色设定

### Q: 如何贡献新的风格？

参阅 [开发指南](DEVELOPMENT.md) 中的"如何添加新风格"章节。

### Q: 支持哪些浏览器？

- Chrome 90+
- Firefox 90+
- Safari 15+
- Edge 90+

所有现代浏览器都支持。部分高级 CSS 效果（如 backdrop-filter）在旧版浏览器中会优雅降级。

### Q: 移动端体验如何？

完全响应式设计，支持：
- 手机竖屏（375px+）
- 平板横屏/竖屏
- 桌面端各种尺寸

搜索框、筛选面板、卡片网格都会自适应屏幕宽度。
