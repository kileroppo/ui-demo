## 常见使用场景

### 场景化入口

首页提供三种场景化入口卡片，适合不同需求的用户：

| 入口 | 适用场景 | 跳转目标 |
|------|----------|----------|
| 我知道我要什么 | 已有明确风格方向 | 风格库搜索 |
| 帮我选 | 不确定选什么风格 | 风格顾问问卷 |
| 我有参考图 | 有设计参考图片 | 图片匹配页面 |

### 我是设计师

**目标：** 寻找设计灵感，确定项目视觉方向。

**推荐流程：**

1. 打开首页，选择"帮我选"进入风格顾问，或直接浏览精选风格卡片
2. 点击感兴趣的风格卡片，进入详情页查看完整的 CSS 实时演示
3. 在详情页中关注以下信息：
   - **全尺寸 Demo** - 切换 Desktop/Tablet/Mobile 视口查看完整页面效果
   - **组件预览** - 查看 Button/Card/Form/Nav 在该风格下的表现
   - **颜色方案** - 主色、辅色搭配建议
   - **适用场景** - 这种风格最适合什么类型的产品
4. 找到心仪风格后，使用导出功能获取 CSS Variables 或 Markdown 摘要
5. 或前往提示词工作台（`/workshop`）自定义模板变量，生成更精确的提示词

**快速技巧：**
- 使用 `Cmd/Ctrl + K` 快速打开搜索
- 点击爱心图标收藏喜欢的风格
- 选择多个风格进行并排对比
- 上传参考图快速匹配风格

### 我是前端开发者

**目标：** 确定技术方案，了解各风格的实现复杂度。

**推荐流程：**

1. 访问风格库页面（`/styles`），使用筛选面板按性能等级排序
2. 关注风格详情中的技术参数：
   - **性能等级** - High/Medium/Low，影响页面加载速度
   - **无障碍等级** - AA/AAA，影响 WCAG 合规
   - **CSS 复杂度** - 通过实时演示代码评估实现难度
3. 查看组件预览（Button/Card/Form/Nav）了解风格化实现
4. 导出 CSS Variables 文件，直接在项目中使用
5. 使用提示词工作台生成 Code Gen 提示词，让 AI 辅助生成组件代码

**快速技巧：**
- 在筛选面板中选择 "High Performance" 筛选高性能风格
- 使用对比功能（`/compare`）对比不同风格的性能参数
- 访问风格时间线了解设计趋势变化
- 创建"我的项目"保存技术选型方案

### 我是产品经理

**目标：** 为产品选择合适的设计风格，与设计师沟通设计方向。

**推荐流程：**

1. 使用首页"帮我选"入口进入风格顾问（`/advisor`）
2. 回答 5 个简单问题，获得个性化风格推荐
3. 或直接访问产品推荐页面（`/products`），找到你的行业分类
4. 选定风格后，创建"我的项目"（`/projects`）保存决策
5. 导出 Markdown 摘要分享给团队

**快速技巧：**
- 产品页面按行业分组，可以直接定位到你的行业
- 使用风格对比功能向领导或团队展示选型对比
- 风格时间线帮助了解设计趋势

## 筛选策略

### 按性能选择

风格库页面的筛选面板支持按性能等级筛选：

- **High Performance** - CSS 实现简单，渲染性能好，适合移动端优先的项目
- **Medium Performance** - 使用了一些高级 CSS 特性（如 backdrop-filter），在现代浏览器上表现良好
- **Low Performance** - 包含复杂动画或大量图层叠加，建议只用于桌面端或展示型页面

**建议：**
- 电商类产品优先选择 High Performance 风格
- 品牌展示页面可以选择 Medium/Low Performance 但视觉冲击力更强的风格
- 移动端应用建议限定在 High/Medium Performance 范围

### 按无障碍等级选择

- **AAA** - 最高无障碍标准，对比度和可读性最佳，适合政府、教育、医疗类产品
- **AA** - 满足基本无障碍要求，适合大部分商业产品
- **A** - 基本对比度，可能需要额外调整以满足合规要求

**建议：**
- 面向公众的政务/教育产品，建议选择 AAA 等级
- 商业 SaaS 产品，建议至少选择 AA 等级
- 纯展示型或艺术类项目可以适当放宽

### 按行业选择

访问产品推荐页面，系统已按 8 大行业预分组：

| 行业 | 典型产品 | 推荐风格方向 |
|------|----------|--------------|
| 科技 SaaS | CRM、项目管理、开发工具 | 极简、扁平、暗色 |
| 电商零售 | 商城、品牌官网 | 鲜明色彩、卡片式 |
| 医疗健康 | 健康 App、医疗平台 | 干净、高对比度 |
| 金融保险 | 银行、投资、保险 | 稳重、专业、高可信度 |
| 教育文化 | 在线课程、知识库 | 友好、易读、色彩温和 |
| 生活服务 | 餐饮、旅行、本地生活 | 活泼、图片丰富 |
| 创意设计 | 设计工具、Portfolio | 前卫、视觉冲击力 |
| 其他 | 更多类型 | 因产品而异 |

## AI 提示词使用方法

### 提示词分段

风格详情页的提示词分为三个标签页：

| 标签 | 用途 | 适用工具 |
|------|------|----------|
| UI Design | 界面设计提示词 | ChatGPT、Claude、通义千问 |
| Image Gen | 图片/视觉生成提示词 | Midjourney、DALL-E、Stable Diffusion |
| Code Gen | 代码生成提示词 | Cursor、GitHub Copilot、Claude |

### 支持的 AI 工具

本项目生成的提示词适用于以下 AI 工具：

- **ChatGPT (GPT-4/4o)** - 生成设计描述、代码
- **Claude** - 生成设计方案、组件代码
- **通义千问** - 中文场景下的设计建议
- **Midjourney / DALL-E** - 生成设计参考图
- **Cursor / GitHub Copilot** - 辅助生成组件代码

### 如何复制提示词

1. 在风格详情页选择需要的提示词分段（UI Design / Image Gen / Code Gen）
2. 点击复制按钮（带有复制图标）
3. 按钮会显示"已复制"确认动效
4. 打开你的 AI 工具，直接粘贴使用

### 如何修改提示词获得更好效果

复制到的提示词是通用模板，你可以根据需要修改：

**添加具体约束：**
```
[粘贴的提示词]

额外要求：
- 主色调使用品牌色 #1a73e8
- 适配移动端 375px 宽度
- 使用 Tailwind CSS 实现
- 包含暗色模式变体
```

**指定具体组件：**
```
[粘贴的提示词]

请基于这个风格，为我设计一个：
- 登录页面
- 数据仪表盘
- 产品定价卡片
- 导航栏组件
```

**组合多种风格：**
```
主要风格：[风格A的提示词]
辅助风格：[风格B的提示词]

请融合这两种风格的特点，设计一个现代感的 SaaS 产品首页。
```

## 提示词工作台

提示词工作台（`/workshop`）提供可视化的模板变量编辑器，帮助你生成更精确的定制化提示词。

### 使用方法

1. 访问 `/workshop` 页面
2. 从模板列表中选择一个风格模板
3. 在编辑区域修改模板变量：
   - **品牌色** - 输入你的品牌主色
   - **字体** - 指定使用的字体
   - **布局类型** - 选择页面布局方式
   - **其他变量** - 根据模板不同而变化
4. 右侧实时预览区会即时更新生成的完整提示词
5. 满意后点击复制按钮，将完整提示词复制到剪贴板

### 适用场景

- 需要为特定品牌定制风格提示词
- 想要批量生成不同配色方案的提示词
- 需要将风格模板与具体项目参数结合

## 图片匹配风格

图片匹配功能（`/image-match`）允许你上传设计参考图，系统自动分析图片主色调并匹配最接近的 UI 风格。

### 使用方法

1. 访问 `/image-match` 页面
2. 点击上传区域或拖拽图片上传
3. 系统自动处理图片：
   - 提取图片主色调（dominant colors）
   - 分析色彩分布和对比度
   - 与 76 种风格的配色方案逐一比对
4. 查看匹配结果：
   - 显示匹配度最高的风格列表
   - 展示提取的主色调色板
   - 点击风格名可跳转到详情页

### 支持的图片格式

- PNG、JPG/JPEG
- 建议上传清晰的 UI 截图或设计稿

### 使用建议

- 上传的图片最好是 UI 界面截图，而非自然照片
- 包含明确配色方案的图片匹配效果更好
- 可以上传竞品截图来找到类似的设计风格

## 我的项目管理

"我的项目"（`/projects`）功能帮助你将风格选择与实际项目关联，集中管理设计决策。

### 创建项目

1. 访问 `/projects` 页面
2. 点击"新建项目"按钮
3. 填写项目信息：
   - **项目名称** - 为项目起一个易识别的名字
   - **绑定风格** - 从 76 种风格中选择主风格
   - **颜色方案** - 自定义品牌配色
   - **字体选择** - 指定项目使用的字体

### 管理项目

- **编辑** - 随时修改项目的风格/颜色/字体绑定
- **删除** - 移除不再需要的项目
- **生成提示词** - 基于项目配置自动生成定制化 AI 提示词

### 数据存储

所有项目数据保存在浏览器 localStorage 中，无需登录即可使用。注意：
- 清除浏览器数据会丢失项目信息
- 不同浏览器/设备间数据不互通

## 设计方案导出

在风格详情页可以将风格的设计方案导出为文件，方便在项目中直接使用或分享给团队。

### 导出 CSS Variables

点击导出按钮，选择"CSS Variables"格式：

```css
:root {
  --color-primary: #hexcolor;
  --color-secondary: #hexcolor;
  --color-accent: #hexcolor;
  --border-radius: ...;
  --shadow: ...;
  /* 更多变量... */
}
```

导出的文件可以直接在项目中引入使用。

### 导出 Markdown 摘要

选择"Markdown"格式导出，生成包含以下内容的文档：

- 风格名称和描述
- 配色方案说明
- 适用场景建议
- 技术参数（性能等级、无障碍等级）
- 推荐产品类型
- AI 提示词

适合作为设计决策文档分享给团队成员。

## 风格对比

风格对比功能（`/compare`）允许你选择 2-3 种风格进行并排对比，方便做出最终决策。

### 使用方法

1. 在风格库页面（`/styles`）浏览风格
2. 点击风格卡片上的"对比"按钮，将风格加入对比栏
3. 底部会出现浮动的 CompareBar，显示已选风格
4. 选择 2-3 种风格后，点击 CompareBar 中的"开始对比"按钮
5. 进入对比页面（`/compare`），并排查看：
   - 视觉演示效果
   - 配色方案对比
   - 性能等级对比
   - 无障碍等级对比
   - 适用场景对比

### 注意事项

- 最少选择 2 种，最多选择 3 种风格
- 可以随时从 CompareBar 中移除已选风格
- 对比页面支持响应式布局

## 风格演变时间线

风格时间线（`/timeline`）以可视化方式展示 UI 设计风格的历史演变过程。

### 时间线节点

| 年份 | 风格 | 说明 |
|------|------|------|
| 2007 | Skeuomorphism | 拟物化设计，模拟真实物体质感 |
| 2013 | Flat Design | 扁平设计，去除多余装饰 |
| 2014 | Material Design | 质感设计，纸张隐喻与动效 |
| 2020 | Neumorphism | 新拟态，柔和凸起效果 |
| 2020 | Glassmorphism | 玻璃拟态，模糊透明效果 |
| 2023 | Bento Grid | 便当盒布局，模块化网格 |
| 2025 | Liquid Glass | 液态玻璃，流动质感 |

### 交互方式

- 沿时间轴滚动浏览各个时代的设计风格
- 点击任意节点跳转到对应风格的详情页面
- 了解每种风格的历史背景和设计理念演变

## 暗色模式

### 切换方法

- 点击页面顶部导航栏中的主题切换按钮（太阳/月亮图标）
- 首次访问时自动跟随系统偏好设置

### 行为说明

- 切换偏好保存在 localStorage 中，下次访问自动恢复
- 所有页面和组件都支持暗色模式
- 风格演示区域保持独立配色，不受暗色模式影响

## 收藏功能

### 使用方法

1. 在风格库或首页中，将鼠标悬停在风格卡片上
2. 点击卡片右上角的爱心图标
3. 图标变为实心红色，表示已收藏

### 管理收藏

- 收藏数据持久化存储在 localStorage 中
- 再次点击爱心图标可取消收藏
- 收藏状态在所有页面同步

## FAQ 常见问题

### Q: 风格数据多久更新一次？

当前版本包含 76 种风格和 161 种产品类型。数据来源于 UI/UX Pro Max 技能库，会随着上游数据更新而更新。

### Q: 可以离线使用吗？

可以。运行 `pnpm build` 构建后，`dist/` 目录下是纯静态文件，可以直接用任何 HTTP 服务器托管，不需要后端服务。

### Q: 搜索没有结果怎么办？

- 尝试用英文关键词搜索（如 "glassmorphism" 代替 "玻璃拟态"）
- 减少搜索关键词长度，使用更通用的词
- 使用筛选面板代替搜索来浏览所有风格
- 查看搜索建议中的热门推荐

### Q: 如何判断一种风格是否适合我的项目？

参考以下维度：
1. **行业匹配** - 在产品推荐页面查看你的行业推荐
2. **性能要求** - 移动端优先选 High Performance
3. **用户群体** - 面向老年人/残障人士选择 AAA 无障碍等级
4. **品牌调性** - 科技感选赛博朋克/暗色，亲和力选圆角/渐变
5. **设计趋势** - 参考风格时间线了解当前趋势

### Q: AI 提示词生成的效果不理想怎么办？

- 使用提示词工作台（`/workshop`）自定义模板变量
- 在提示词后追加具体约束（颜色、尺寸、用途）
- 选择正确的提示词分段（UI Design / Image Gen / Code Gen）
- 尝试组合两种风格的提示词
- 在提示词开头加上"你是一个资深 UI 设计师"的角色设定

### Q: 项目数据会丢失吗？

"我的项目"数据存储在浏览器 localStorage 中。以下情况会导致数据丢失：
- 清除浏览器缓存/数据
- 使用无痕/隐私浏览模式
- 切换不同浏览器或设备

建议重要项目通过导出功能保存 Markdown 摘要作为备份。

### Q: 对比功能最多可以选几个风格？

最多选择 3 种风格进行对比。至少需要选择 2 种才能进入对比页面。

### Q: 图片匹配的准确度如何？

图片匹配基于颜色分析算法，对配色方案明确的 UI 截图效果较好。自然照片或配色不明确的图片匹配效果可能不理想，建议上传 UI 界面截图。

### Q: 如何贡献新的风格？

参阅 [开发指南](DEVELOPMENT.md) 中的"如何添加新风格"章节。

### Q: 支持哪些浏览器？

- Chrome 90+
- Firefox 90+
- Safari 15+
- Edge 90+

所有现代浏览器都支持。部分高级 CSS 效果（如 backdrop-filter）在旧版浏览器中会优雅降级。

### Q: 移动端体验如何？

完全响应式设计，支持：
- 手机竖屏（375px+）
- 平板横屏/竖屏
- 桌面端各种尺寸

搜索框、筛选面板、卡片网格都会自适应屏幕宽度。底部导航栏在移动端自动切换为底部标签栏。
